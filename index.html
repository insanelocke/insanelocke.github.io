REGLAMENTO OFICIAL

LAS REGLAS

CAPTURA POR RUTA
// REGLA 01 //
  • Solo se puede capturar el primer Pokémon que aparezca en cada ruta
  • Si el Pokémon es debilitado o huye, se pierde la captura en esa ruta definitivamente
  • No se permiten intentos adicionales bajo ninguna circunstancia
MUERTE PERMANENTE
// REGLA 02 //
  • Un Pokémon debilitado se considera muerto de forma permanente
  • Deberá ser almacenado en el PC sin posibilidad de volver al equipo
  • Solo puede usarse para Movimientos Ocultos (MO) si es necesario para progresar
CONDICIÓN DE DERROTA
// REGLA 03 //
  • El desafío se pierde cuando los 6 Pokémon del equipo principal quedan debilitados simultáneamente
  • Esto aplica independientemente de los Pokémon disponibles en el PC
SISTEMA DE VIDAS
// REGLA 04 //
  • El jugador comienza con 20 vidas en total
  • Cada muerte de un Pokémon consume 1 vida
  • Si las vidas llegan a 0, el desafío finaliza de forma definitiva
  • La vida 0 también cuenta como derrota
RESTRICCIÓN DE OBJETOS
// REGLA 05 //
  • Tras cada líder de gimnasio, máximo 1 poción adquirida
  • Compra libre permitida únicamente para: Poké Balls y Repelentes
  • Cualquier otro objeto queda restringido salvo indicación contraria
RANDOMIZACIÓN TOTAL
// REGLAS 06 & 07 //
  • Pokémon legendarios: prohibidos bajo cualquier circunstancia
  • El juego está completamente randomizado en todos sus sistemas
  • Tipos de Pokémon, movimientos, evoluciones: todo alterado
  • Cualquier Pokémon puede tener tipos inesperados o evolucionar en formas distintas
  • Todo conocimiento previo del juego queda invalidado

Cualquier regla adicional olvidada durante la preparación o desarrollo del desafío podrá ser añadida o modificada en futuras actualizaciones del reglamento oficial.
